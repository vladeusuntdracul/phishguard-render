# Quick Start Guide

## Deploy on Replit (FREE)
1. Go to replit.com
2. Create Python repl
3. Upload all files
4. Click Run

## Deploy Locally
1. Extract files
2. Run: pip install -r requirements.txt
3. Run: streamlit run app.py --server.port 5000
